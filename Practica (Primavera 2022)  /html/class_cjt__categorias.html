void Cjt_categorias::leer_cjt_categorias ( )

Lee un conjunto de categorías.

Precondición
Hay preparados en el canal estandar: el número de categorías C, el número maximo de niveles K, una secuencia de C strings. Seguidamente, C x K enteros (>= 0).
Postcondición
El parámetro implícito pasa a tener C categorias identificadas por nombre con K máximos niveles y su matriz de puntos correspondiente de CxK enteros.
